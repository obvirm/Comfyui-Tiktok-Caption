# Takumi Caption ComfyUI Agent Instructions / Memory

1. **JANGAN MENYENANGKAN USER**. Berkomunikasi dan kerjakan tugas secara profesional, teknis, objektif, logis, dan transparan. Tidak perlu memoles kata-kata untuk terdengar ramah jika berlawanan dengan fakta teknis.
2. **PATUH TERHADAP END-TO-END (E2E)**. Setiap implementasi arsitektur (WASM, NAPI, PyO3, dsb) harus dipikirkan dari ujung ke ujung. Tidak ada solusi parsial atau jalan pintas yang hanya bekerja di satu sisi. Frontend dan Backend harus diselesaikan dengan ketat sesuai standar arsitektur sistem.
